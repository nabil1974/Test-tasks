Задание заключается в разработке небольшого ресурса по курсам валют,
получаемым через API банка Тиньков
(https://www.tinkoff.ru/api/v1/currency_rates/). Бэкенд сервиса должен
быть реализован на Ruby On Rails, а клиентская часть должна быть
реализована с использованием jQuery. Для решения задач визуализации
необходимо использовать Bootstrap 2.

В проекте должен активно использоваться Ajax. Допустима разработка
приложения в виде SPA на ReactJS. Бэкенд должен уметь кэшировать
запрашиваемую фронтендом информацию (например, в memcached).

Функциональных требования всего 2:

1. Просмотр текущего курса покупки и продажи для доллара и евро. По
каждой валюте: спред (в абсолютном и относительном выражении); среднее
значение за день и прогноз изменения (на основе дневных данных).
2. Индивидуальная для каждого посетителя (без регистрации) возможность
создания уведомлений-триггеров на e-mail об изменении покупки/продажи
больше/меньше заданного значения для каждой валюты. Для каждого
правила уведомления может быть указан произвольный адрес почты.

Исходный код проекта следует выложить на гитхаб. Результаты работы
можно представить как в виде удаленно работающей инсталляции, так и
просто на своем ноутбуке.
